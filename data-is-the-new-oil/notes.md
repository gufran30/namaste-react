## Higher Ordr Component

- HOC is a function that takes a component and return a component.
- at the end its a js function.
- it takes component as an input and enhance the component and return it

### Why we use HOC ?

- HOCs are a design pattern used for code reuse and abstracting common behaviors, like adding styling, handling authentication, or managing loading states, across multiple components.

### React application have two layer

1. #### Data Layer

   - consist of state, props,etc.

2. #### UI Layer

   - UI layer is powered by data layer
